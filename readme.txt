This widget shows the count of Facebook, Twitter, Google Plus activities for your domain. 

== Description ==

This widget shows the number of Facebook likes, shares and comments, Twitter tweets and Google+1's for your Wordpress installation's domain.

== Installation ==

1. Download the Social Counter Dashboard Widget .
2. In your WordPress Admin, go to Plugins > Add New > Upload, and select the plugin ZIP file.
3. Activate the plugin
